# react-redux-infrastructure

Library contains a set of functions and pre-installed modules that help when working with Redux State Manager.

> Library is still in an early development stage :smile:

## Installing

Install with `npm`:

```sh
$ npm install
```

Install with `yarn`:

```sh
$ yarn
```
